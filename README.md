实现的功能：

管理员、商家、用户三种角色，管理员和商家是PC后台，用户是小程序端

管理员可以管理用户、分类、商品、订单等

商家可以上传商品、查看订单、发货等

用户可以下单购买商品、评价等

用到的技术：

后台 JAVA语言的SpringBoot框架、MySql数据库、Maven等；

前台 Vue语法的UniApp框架，可以发布成微信小程序到微信开发者平台
